# Ecommerce Website

A basic ecommerce frontend application built with React, TypeScript, and Vite.
The project allows users to browse products, search and filter products, place orders, and track deliveries.

## Features

* Display all available products
* Server-side product filtering
* Product search functionality
* Checkout process
* Order management
* Order tracking
* Responsive user interface
* Page and component testing using Vitest and Testing Library

## Pages

* **Home Page** - Displays all available products
* **Checkout Page** - Handles cart and order checkout process
* **Order Page** - Displays user order information
* **Tracking Page** - Tracks order delivery status

## Tech Stack

## Installation

Clone the repository:

```bash
git clone https://github.com/Info-Bash/ecommerce-app.git
```

Navigate into the project folder:

```bash
cd ecommerce-website
```

Install dependencies:

```bash
npm install
```

## Running the Project

Start the development server:

```bash
npm run dev
```

## Build Project

Build the frontend and backend:

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Running Tests

Run tests with Vitest:

```bash
npx vitest
```
